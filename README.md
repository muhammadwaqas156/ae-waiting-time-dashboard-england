# A&E Waiting Time and Attendance Dashboard — England (2004–2026)

## Overview

This project presents an interactive Excel dashboard analyzing Accident & Emergency (A&E) performance in England using NHS data.

The dashboard tracks patient attendance trends and measures the percentage of patients seen within the NHS 4-hour target.

## Key Features

• KPI indicators for latest performance, attendance, and waiting times
• Trend analysis of total A&E attendances
• Performance monitoring against NHS standards
• Interactive filtering by quarter and year

## Tools Used

• Microsoft Excel
• Pivot Tables
• Pivot Charts
• Dashboard Design

## Key Insights

• A&E attendances increased significantly over time
• Performance declined to 74% in the latest period
• Results fall below the NHS 95% target



## Data Source

NHS England
